# 💛 Flores Amarillas — Para mi Corazón

Una página web romántica creada con amor puro en HTML, CSS y JavaScript.

## ✨ Características

- 🌟 **4 pantallas** con transiciones suaves
- 💌 **Sobre animado** que se abre al hacer click
- ✍️ **Efecto typewriter** en la carta romántica
- 🌸 **9 tarjetas de razones** interactivas que revelan mensajes
- 🎆 **Fuegos artificiales** dorados en la pantalla final
- 🌻 **Flores y girasoles** cayendo en el fondo
- ⭐ **Fondo de estrellas** animado con Canvas
- 🖱️ **Cursor personalizado** dorado
- 📱 **Soporte para swipe** en móvil
- ⌨️ Navegación con **teclas de flecha**

## 🚀 Cómo usar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/flores-amarillas.git
   ```

2. Abre `index.html` en tu navegador — ¡no necesita servidor!

3. O activa **GitHub Pages** para compartir el link.

## 📁 Estructura

```
flores-amarillas/
├── index.html   ← Estructura de la página
├── style.css    ← Estilos elegantes y animaciones
├── script.js    ← Toda la magia interactiva
└── README.md    ← Este archivo
```

## 🌐 Publicar en GitHub Pages

1. Sube los archivos a un repositorio público en GitHub
2. Ve a **Settings → Pages**
3. En *Source*, selecciona `main` branch → `/root`
4. ¡Copia el link y mándaselo a ella! 💛
